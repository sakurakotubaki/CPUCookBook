条件分岐と繰り返し
プログラムの流れには、「順次進行」「条件分岐」「繰り返し」の３種類があります。

- 順次進行とは、条件に応じて任意のアドレスの命令を実行することです。
- 条件分岐とは、条件に応じて任意のアドレスの命令を実行することです。
- 繰り返しとは、同じアドレスの命令を何度か繰り返し実行することです。

## プログラムカウンターの例
順次進行の場合は、プログラムカウンタの値が１ずつ増加。
条件分岐や繰り返しがある場合は、マシン語命令が、プログラム・カウンタの値を任意のアドレスに + 1でない値に設定することになる。
これによって前のアドレスに戻って同じ命令を繰り返したり、任意のアドレスにジャンプして分岐したりできる。
これをプログラムにしてください。
